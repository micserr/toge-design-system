# Installation

> [!IMPORTANT]
> Before installing ***Design System Next***, make sure you're connected to the ***Azure Artifacts feed***. Connecting to the feed is **required to authenticate and access our package**, as it is currently **private**. For instructions on setting up the connection, see [Connect to Feed](https://dev.azure.com/sproutphil/Sprout%20Design%20System/_artifacts/feed/Design-System-Next/connect)

Once connected, install the package via npm:

```bash
npm install design-system-next
```
