# ตัวอักษร

เราได้สร้างแบบแผนตัวอักษรเพื่อให้มั่นใจถึงการนำเสนอที่สอดคล้องและเหมาะสมในทุกแพลตฟอร์ม
ตัวเลือกเหล่านี้สะท้อนถึงเอกลักษณ์ภาพและหลักการออกแบบที่ชี้นำ UI/UX ของเรา เพื่อให้มั่นใจถึงความชัดเจนและความง่ายในการอ่าน

## ครอบครัวตัวอักษร

ครอบครัวตัวอักษรกำหนดตัวอักษรหลักที่ใช้ในการออกแบบผลิตภัณฑ์ของเรา
การใช้ครอบครัวตัวอักษรอย่างสม่ำเสมอในทุกแพลตฟอร์มจะรักษาความสอดคล้องของแบรนด์และความสามารถในการอ่าน
ครอบครัวตัวอักษรต่อไปนี้ใช้ในระบบตัวอักษรของเรา:

<p>- เป็นตัวอักษรหลักที่ใช้สำหรับข้อความ UI ทั่วไป มีการออกแบบที่ทันสมัยและโค้งมนเพื่อให้มั่นใจถึงความชัดเจนและความสามารถในการอ่านที่ราบรื่น</p>
